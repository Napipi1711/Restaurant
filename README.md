# Restaurant Management System

This project is a full-stack Restaurant Management System designed to support multiple platforms including web and mobile applications. It helps restaurants manage orders, payments, and customer interactions efficiently.

## Technologies Used

### Backend
- JavaScript (Node.js)
- RESTful API architecture

### Frontend (Web)
- React.js
- Responsive and modern UI

### Mobile Applications
- Dart (Flutter framework)
- Includes:
  - `frontend_app`: Customer mobile application
  - `POS`: Point of Sale application for staff

## Project Structure

- `backend/` – Handles business logic, APIs, and database operations
- `frontend/` – Web interface built with React.js
- `frontend_app/` – Mobile app for customers (Flutter/Dart)
- `POS/` – POS system for restaurant staff (Flutter/Dart)

## Features

- User authentication (login system)
- Order management
- Cash payment support
- Cross-platform usage (Web & Mobile)
- Real-time data handling

## Ongoing Development

- Revenue Dashboard (for tracking sales and analytics) is currently under development.
