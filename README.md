Restaurant Management System

This project is a full-stack Restaurant Management System designed to support multiple platforms including web and mobile applications. It helps restaurants manage orders, payments, and customer interactions efficiently.

### Project Evolution
Phase 1 (2024 – 2025)

In the initial phase, the system was developed using:

C# (ASP.NET MVC) for backend and web application
SQL Server for database management
MVC architecture for organizing the system structure

During this stage, the project focused on fundamental features such as:

User authentication
Basic order management
Database design and relationships
Simple web interface

The goal of this phase was to build a solid foundation in backend development, database handling, and system architecture.

### Phase 2 (2025 – 2026)

The system was upgraded into a modern full-stack and cross-platform application using:

JavaScript (Node.js) for backend
React.js for web frontend
Flutter (Dart) for mobile applications

## This phase significantly improved the system by:

Supporting both Web and Mobile platforms
Enhancing UI/UX with modern design
Implementing RESTful APIs
Handling real-time data
Expanding system scalability
.Technologies Used:Backend , JavaScript (Node.js) , RESTful API architecture .Frontend (Web)React.js .Responsive and modern UI
Mobile Applications : Dart (Flutter framework)
Includes:
frontend_app: Customer mobile application
POS: Point of Sale application for staff

## Project Structure
backend/ – Handles business logic, APIs, and database operations
frontend/ – Web interface built with React.js
frontend_app/ – Mobile app for customers (Flutter/Dart)
POS/ – POS system for restaurant staff (Flutter/Dart)

## Features
User authentication (login system)
Order management
Cash payment support
Cross-platform usage (Web & Mobile)
